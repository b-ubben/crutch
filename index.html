Crutch Typography Layout Forms and Input Utilities

Welcome to Crutch.

Crutch is a simple and clean CSS framework offering straightforward, responsive front-end development in a smaller and easier to understand package than other similar frameworks. Crutch uses a simple three-column grid system for layout and comes with a host of utilities for margin, padding, positioning, and flexbox classes as well as default styling for many HTML elements.

Typography

Crutch's typography styling is appealing and modern. Several text-manipulation classes are also included for the user as well.

Headings appear as follows: .h1 .h2 .h3 .h4 .h5 .h6

In addition to styling for headings, Crutch includes special classes for large text display.

These classes are:
.display-medium
.display-big
.display-bigger
.display-biggest

Text color and visibility can also be controlled via these classes: .text-dark .text-light .text-hide

Manipulation of text is acheived through these classes:
.text-transform-lowercase
.text-transform-uppercase
.text-transform-capitalize
.font-weight-normal
.font-weight-bold
.font-weight-italic
.text-left
.text-right
.text-center
.text-justify
.text-white-space-nowrap
.text-truncate
.underline
.strike-through

Layout and Positioning

Crutch uses a simple three column grid system for layout. Simply create a parent container called a .row, and within that row use either .item, .item-half, or .item-third to section off content. At mobile resolutions these items stack upon one-another. To keep these items from doing so, simply add the class .flex-nowrap to the parent row container element.

Included along with layout classes are a few classes to control positiong. These classes are:
.fixed-top
.fixed-bottom
.fixed-left
